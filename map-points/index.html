Map Points

Embed a Google Map

If all you need to do is show locations on a map, then use Google Maps to create a custom map and embed it on your site (the blue marker in the center is a custom one that I made).

Custom maps are part of the My Maps section of your Google account - you can read about creating and customizing maps here.

p5

Not to keep bringing up p5, but there's a reason why it's included as part of this class - it can make many interactions easier to implement. The challenge here is that you will need to define clickable areas yourself, and that might require some trial-and-error for marker placement.

In the example below, I use a map screenshot image with a circle on top to define the location. The collide2D library from p5 helps to detect whether the cursor is in the correct area. Yes, this is a little cludgy, but ultimately, the focus should be on implementing this feature in the best way that you can at this time - you can always find new and better ways later!

Mappa

There is a library that integrates with p5 called Mappa.js. This will work better than a screenshot and provides the added benefit of defining your click locations with latitude/longitude coordinates.

An example is not included with this folder as creating multiple p5 canvases can get a little messy, but the link above will walk you through the process. Making the shapes clickable would be the same as with regular p5 (e.g. using the collide2D library combined with mousePressed).
